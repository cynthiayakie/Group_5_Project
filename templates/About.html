About This Project

This project analyzes hockey statistics to provide insights into team performances, player stats, and trends across seasons.

Dataset Variables

  • Team Name: Identifies the hockey team by name.
  • Year: Specifies the season's year.
  • Wins: Tallies the team's victories.
  • Losses: Counts the team's defeats.
  • OT Losses: Tracks losses in overtime.
  • Win %: Calculates the team's winning ratio.
  • Goals For (GF): Summarizes goals scored by the team.
  • Goals Against (GA): Summarizes goals conceded by the team.
  • + / -: Represents the team's goal differential (Goals For minus Goals Against).

Data Source

The data used in this project comes from a comprehensive hockey statistics dataset, accessed from Scrape This Site.
